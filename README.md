 گزارش پروژه: چشمک‌زن LED با آردوینو
نام نویسنده : ایلیا برنامهر 
اعضای گروه : ایلیا برنامهر ، امین طلوع فکور
 ۱. مقدمه  
کد چشمک زدن ال ای دی در برد بورد با استفاده از یک برد آردوینو.


 ۳. قطعات مورد استفاده  
- برد آردوینو Uno  
 چراغ ال ای دی 
- مقاومت ۲۲۰ اهم (رنگ‌بندی: قرمز-قرمز-قهوه‌ای)  
- برد برد  
- سیم‌های جامپر(سیم دو سرنری)

 ۴. نحوه‌ی ساخت مدار  
مدار به صورت زیر متصل شده است:  
- پایه‌ی مثبت LED (پایه‌ی بلندتر) به پین دیجیتال شماره ۸ آردوینو وصل شده است.  
- پایه‌ی منفی LED (پایه‌ی کوتاه‌تر) از طریق یک مقاومت ۲۲۰ اهم به GND (زمین) آردوینو متصل شده است.  
- مقاومت برای جلوگیری از سوختن LED و محدود کردن جریان عبوری استفاده شده اس.

ک5.کد مورد استفاده:


void setup() {
  pinMode(8, OUTPUT);   // تنظیم پین ۸ به عنوان خروجی
}

void loop() {
  digitalWrite(8, HIGH);  // روشن کردن LED
  delay(500);             // تأخیر ۵۰۰ میلی‌ثانیه
  digitalWrite(8, LOW);   // خاموش کردن LED
  delay(500);             // تأخیر ۵۰۰ میلی‌ثانیه
}


 
 ۶. نتایج  
پس از آپلود کد به آردوینو، LED یک بار در هر ثانیه چشمک می‌زند. این رفتار نشان‌دهنده‌ی عملکرد صحیح مدار و کد است.

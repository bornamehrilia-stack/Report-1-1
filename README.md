### ۱. هدف آزمایش

آشنایی با ساختار برد **Arduino Uno**، نحوه تعریف پین‌های خروجی و تولید پالس‌های زمانی (Delay) برای کنترل یک قطعه الکترونیکی (LED).

### ۲. شرح مختصر مدار

در این مدار، یک LED به عنوان خروجی به پین شماره ۸ برد آردوینو متصل شده است. برنامه نوشته شده وظیفه دارد ولتاژ این پین را در بازه‌های زمانی مشخص بین وضعیت منطقی ۱ (High) و ۰ (Low) تغییر دهد تا حالت چشمک‌زن ایجاد شود. برای جلوگیری از سوختن LED، یک مقاومت محدودکننده جریان نیز در مسیر مدار قرار گرفته است.

### ۳. موارد مورد نیاز

* ۱ عدد برد **Arduino Uno**
* ۱ عدد **LED** 
* ۱ عدد **مقاومت** 
* برد بورد و سیم‌های اتصال (Jumper Wires)
* کابل USB جهت پروگرام کردن

### ۴. نمایش کد پروژه

```cpp
void setup() { 
  // تنظیم پین شماره 8 به عنوان خروجی
  pinMode(8, OUTPUT); 
}

void loop() {
  digitalWrite(8, HIGH); // روشن کردن LED (ارسال 5 ولت)
  delay(500);            // توقف برنامه به مدت 500 میلی‌ثانیه
  digitalWrite(8, LOW);  // خاموش کردن LED (ارسال 0 ولت)
  delay(500);            // توقف برنامه به مدت 500 میلی‌ثانیه
}

```

### ۵. توضیح مختصر کد

* **تابع `setup`:** این بخش فقط یک‌بار در ابتدای برنامه اجرا می‌شود. در اینجا پین شماره ۸ با دستور `pinMode` به عنوان خروجی دیجیتال معرفی شده است.
* **تابع `loop`:** این بخش به صورت بی‌نهایت تکرار می‌شود. با دستور `digitalWrite` ولتاژ پین ۸ تغییر می‌کند.
* **تابع `delay(500)`:** یک وقفه نیم‌ثانیه‌ای (۵۰۰ میلی‌ثانیه) ایجاد می‌کند تا تغییر حالت LED با چشم انسان قابل تشخیص باشد. در مجموع، هر سیکل کامل روشن و خاموش شدن ۱ ثانیه طول می‌کشد.
